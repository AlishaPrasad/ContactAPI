My Contacts(R) Version 1.0 - 28/01/2018

GENERAL USAGE NOTES
--------------------
- My Contacts can be used to add, update, view and delete information about a contact
- Unit test cases can be used to verify the expected functionality
---------------------------------------------------------------------------------------------------------

Installation instructions
--------------------------
- You will require .Net Core 2.0 and SQL Server 2012
- You can use Visual Studio 2017 to run this project
- Make sure to change the connection string in appsettings.json
- Unit Test cases are available in MyContactsTests project
